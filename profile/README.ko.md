## Vibe Island이란

Vibe Island는 MacBook 노치를 AI 코딩 에이전트의 컨트롤 센터로 바꿉니다. Claude Code, Codex, Gemini CLI, Cursor, OpenCode, Droid, Qoder, Copilot, CodeBuddy, Kiro를 한 곳에서 관리 — 권한 승인, 질문 응답, 정확한 터미널 세션으로 점프, 모든 것을 흐름을 깨지 않고.

- **제로 설정** — 처음 실행 시 지원하는 모든 CLI의 훅을 자동 구성
- **정확한 점프** — 한 번의 클릭으로 iTerm2, Ghostty, Warp, Terminal.app, VS Code, Cursor 등의 정확한 탭과 분할 창으로 이동
- **완전 로컬** — Unix 소켓 통신, 클라우드 없음, 계정 없음, 텔레메트리 없음
- **네이티브 Swift** — RAM 50MB 미만, 유휴 시 CPU 거의 제로, Apple Silicon 네이티브 빌드
