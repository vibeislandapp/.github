## What is Vibe Island

Vibe Island transforms your MacBook notch into a control center for AI coding agents. Watch Claude Code, Codex, Gemini CLI, Cursor, OpenCode, Droid, Qoder, Copilot, CodeBuddy, and Kiro — approve permissions, answer questions, and jump back to the exact terminal session, all without leaving flow.

- **Zero-config** — auto-configures hooks on first launch for every supported CLI
- **Precise jump** — one click lands on the exact tab and split pane across iTerm2, Ghostty, Warp, Terminal.app, VS Code, Cursor, and more
- **Fully local** — all communication over Unix sockets, no cloud, no accounts, no telemetry
- **Native Swift** — under 50 MB RAM, near-zero CPU when idle, built for Apple Silicon

## Quick Start

