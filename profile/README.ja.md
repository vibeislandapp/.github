## Vibe Island とは

Vibe Island は MacBook のノッチを AI コーディングエージェントのコントロールセンターに変えます。Claude Code、Codex、Gemini CLI、Cursor、OpenCode、Droid、Qoder、Copilot、CodeBuddy、Kiro を一元管理 — 権限を承認し、質問に答え、正確なターミナルセッションに戻る、すべてフローを維持したまま。

- **ゼロ設定** — 初回起動時にすべての対応 CLI のフックを自動設定
- **精密ジャンプ** — ワンクリックで iTerm2、Ghostty、Warp、Terminal.app、VS Code、Cursor などの正確なタブ・分割ペインに移動
- **完全ローカル** — Unix ソケット経由の通信、クラウドなし、アカウントなし、テレメトリなし
- **ネイティブ Swift** — RAM 使用量 50MB 未満、アイドル時 CPU ほぼゼロ、Apple Silicon ネイティブビルド
