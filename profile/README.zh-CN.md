## Vibe Island 是什么

Vibe Island 把 MacBook 的刘海变成一个 AI 编程助手的控制中心。实时监控 Claude Code、Codex、Gemini CLI、Cursor、OpenCode、Droid、Qoder、Copilot、CodeBuddy、Kiro —— 批准权限、回答问题、一键跳回精确的终端会话，整个流程不离开当前工作。

- **零配置** —— 首次启动自动为所有检测到的 CLI 配置 hooks
- **精确跳转** —— 点一下直接跳到 iTerm2、Ghostty、Warp、Terminal.app、VS Code、Cursor 等终端/IDE 的精确 tab 与分屏
- **完全本地** —— Unix socket 通信，无云端、无账号、无遥测
- **原生 Swift** —— 内存占用不到 50MB，空闲时 CPU 近乎零，Apple Silicon 原生构建

## 快速开始

```sh
brew install --cask vibe-island
```

或直接从 [vibeisland.app](https://vibeisland.app) 下载。
